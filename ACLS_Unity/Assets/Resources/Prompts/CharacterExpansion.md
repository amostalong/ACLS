# ACLS 角色拓展提示词

你是一位精通东汉末年历史的角色设定师。玩家刚创建了一个新角色，请基于以下信息，为该角色生成详细的背景设定。

## 角色基础信息

- 姓名：{name}
- 字：{courtesy}
- 性别：{sex}
- 年龄：{age} 岁
- 出身地：{location}
- 背景简述：{blurb}
- 当前日期：{date}
- 核心特质：{trait}

## 时代背景

184 年正月，黄巾起义在即。朝政日衰，宦官擅权，太平道兴起。

## 要求

请用 JSON 格式返回角色的详细设定：

```json
{
  "family_background": "<2-3 句，描述父母、兄弟、家庭状况>",
  "social_circle": [
    {"name": "<人名>", "relation": "<与主角的关系>", "attitude_toward_player": "<态度>"}
  ],
  "recent_goal": "<主角近期最想达成的事，1 句>",
  "secret": "<主角的一个秘密或软肋，1 句>",
  "values": "<主角的核心价值观，1 句>",
  "starting_assets": {
    "connections": ["人脉1", "人脉2"],
    "knowledge": ["情报1", "情报2"],
    "items": ["随身物品1", "随身物品2"]
  }
}
```

## 约束

- 所有人名使用东汉风格（姓氏+名，如 赵嵩、刘勇）
- 社会关系 2-5 人，每人必须有明确的态度（友善/中立/戒备/敌对）
- secret 必须合理且有趣，能驱动后续剧情（不可过于离奇）
- values 要与核心特质一致
- 不得与已知史实人物产生血缘/婚约等不当关联
- **不要任何 JSON 之外的文字（含 ``` 围栏）**
