## 当前任务：构建区域势力层（L3）

这是世界构建的第三步。基于世界观设定（World）和宏观势力格局（L4），你的任务是生成**区域层面的势力与张力**——主角所在大区的权力分布。

## 输出要求（严格）

只输出一个 JSON 对象，包含以下顶层字段（尽量先输出 thinking 字段）：

```json
{
  "thinking": "你的推理过程，尽量先输出",

  "narration": "2~3 句叙事文本，以 DM 口吻向玩家描述所在区域的权力面貌，如「目光收束到你所在的这片土地…」",

  "region": "主角所处大区/地区概括，如「益州·犍为郡一带」",

  "regional_powers": [
    {
      "name": "势力名称",
      "stance": "一句话当前姿态"
    }
  ],

  "regional_tensions": "区域张力/矛盾概括，1~2 句",

  "summary": "区域格局一句话概括，不超过 20 字",

  "chars": [
    {
      "name": "人物名称",
      "role": "身份/职能",
      "location": "所在位置",
      "relation": 0,
      "reachable_in_days": 0
    }
  ],

  "factions": [
    {
      "name": "势力名称",
      "type": "类型",
      "stance": "当前姿态"
    }
  ],

  "places": [
    {
      "name": "地点名称",
      "type": "类型",
      "description": "一句话描述"
    }
  ]
}
```

不要 JSON 之外的任何文字（包括 ``` 围栏）。

- 所有内容必须与世界观设定和宏观格局一致

- 区域张力应反映该地区独有的矛盾
