## 当前任务：构建区域势力层（L3）

这是世界构建的第三步。基于世界观设定（World）和宏观势力格局（L4），你的任务是生成**区域层面的势力与张力**——主角所在大区的权力分布。

## 输出要求（严格）

只输出一个 JSON 对象，包含以下顶层字段（尽量先输出 th 字段）：

```json
{
  "th": "你的推理过程，尽量先输出",

  "nar": "叙事文本，以客观、第三人称的历史旁白笔触呈现这一层的格局与张力。此刻玩家角色尚未登场，禁止用「你」称呼任何人、禁止第二人称。目标是让玩家想继续玩下去——每段叙述都应有足够的吸引力、悬念或张力",

  "reg": "主角所处大区/地区概括，如「益州·犍为郡一带」",

  "rp": [
    {
      "n": "势力名称",
      "st": "一句话当前姿态"
    }
  ],

  "rt": "区域张力/矛盾概括，1~2 句",

  "su": "区域格局一句话概括，不超过 20 字",

  "chars": [
    {
      "n": "人物名称",
      "r": "身份/职能",
      "loc": "所在位置",
      "rel": 0,
      "rid": 0,

      "fa": "父亲姓名",
      "mo": "母亲姓名",

      "cfr": ["核心朋友姓名"],

      "imp": false
    }
  ],

  "fac": [
    {
      "n": "势力名称",

      "st": "当前姿态"
    }

    {
      "n": "地点名称",
      "tp": "类型",
      "desc": "一句话描述"
    }
```
