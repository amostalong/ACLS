## 当前任务：构建世界观（World Setting）

这是世界构建的第一步。你的任务是定义整个游戏世界的**世界观基座**——相当于一个棋盘的棋盘本身，后续所有势力、人物、场景都将建立在这个基座之上。

你的输出将被结构化存储，并作为后续 L4（宏观势力）、L3（区域势力）、L2（近域网络）、L1（当前场景）各层构建的上下文根源。

---

## 玩家输入

[角色描述]
{role_description}

[世界描述]
{world_description}

---

## 规则

- 若玩家提供了具体设定，必须严格沿用，不可擅自修改时代/风格/视角
- 若玩家未提供，由你自由设计，但必须自洽
- 认知边界决定了"这个世界里角色不知道什么"——直接影响叙事中信息的呈现方式
- 历史锚点是未来事件展开的里程碑，需要与时代设定一致

---

## 输出要求（严格）

只输出一个 JSON 对象，包含以下顶层字段（尽量先输出 thinking 字段）：

```json
{
  "thinking": "你的推理过程，尽量先输出",

  "narration": "2~5 句叙事文本，以 DM 口吻向玩家展现这个世界如何展开，让玩家感受到世界在眼前浮现",

  "era_name": "时代全称，如「东汉末年·中平元年（184年）」",

  "narrative_style": "叙事风格描述，1 句话，如「近似古典白话小说，简洁克制」",

  "narrative_perspective": "叙事视角，如「第一人称」或「第三人称有限」",

  "cognitive_boundaries": [
    "这个世界角色不知道的认知限制列表",
    "如：不知细菌与病毒、认为灾异是上天示警、无概率思维"
  ],

  "world_undertones": "当前世界状态的一句话概括，如「黄巾虽平·余波未息；赋税日重·流民四散」",

  "historical_anchors": [
    "3~6 条未来关键历史节点",
    "每条不超过 20 字",
    "如：黄巾覆灭·刘焉入蜀·董卓入京"
  ],

  "world_summary": "一句话世界摘要，不超过 20 字"
}
```

不要 JSON 之外的任何文字（包括 ``` 围栏）。
