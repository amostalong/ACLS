## 当前任务：构建近域网络层（L2）

这是世界构建的第四步。基于世界观设定（World）、宏观势力（L4）和区域格局（L3），你的任务是生成**主角的个人生活圈**——短期触角所及的人脉、势力、地点、事件和机遇。

## 输出要求（严格）

只输出一个 JSON 对象，包含以下顶层字段（尽量先输出 th 字段）：

```json
{
  "th": "你的推理过程，尽量先输出",

  "nar": "叙事文本，以 DM 口吻直接以「你」叙述。目标是让玩家想继续玩下去——每段叙述都应有足够的吸引力、悬念或张力",

  "chars": [
    {
      "n": "人物名称",
      "r": "职能/关系",
      "loc": "所在位置",
      "rel": -50~50 的整数，0 为中立，
      "rid": 按世界观时间单位的可联系距离，0 表示同处一地,

      // 社会关系（可选）
      "fa": "父亲姓名",
      "mo": "母亲姓名",
      "sib": ["兄/弟/姐/妹姓名"],
      "orl": ["其他亲戚姓名"],
      "cfr": ["核心朋友姓名"],

      // 是否重要人物（可选，需要每次更新时重新生成）
      "imp": false
    }
  ],

  "fac": [
    {
      "n": "势力名称",
      "tp": "类型，如望族/行会/商帮/宗门等",
      "st": "当前姿态，一句话"
    }
  ],

  "pl": [
    {
      "n": "地点名称",
      "tp": "类型，如县城/渡口/坞堡/关隘等",
      "desc": "一句话描述"
    }
  ],

  "ev": [
    {
      "ti": "事件标题",
      "urg": "high|medium|low",
      "dl": "期限描述",
      "det": "事件详情"
    }
  ],

  "opp": [
    "机遇1",
    "机遇2"
  ]
}
```

不要 JSON 之外的任何文字（包括 ``` 围栏）。

## 规则

- 所有内容必须与世界观设定和已构建的各层一致
- chars 是主角短期可接触到的人（邻居/同僚/乡绅/商贩/同行等），2~5 人
- fac 是主角生活范围内可见的势力/组织/家族，1~4 个
- pl 是主角生活范围内的关键地点，2~5 个
- ev 是当前正在发生的事件/压力，1~3 条
- opp 是当前可把握的机遇，1~3 条
- 各实体的位置（loc）必须与 L3 的 region 一致
