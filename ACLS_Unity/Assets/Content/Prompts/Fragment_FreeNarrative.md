## 当前任务：推进自由叙事

玩家通过选项或自由输入推进剧情。基于已有的世界、舞台、角色数据进行响应。

**顶层要求：**
- thinking：你的推理过程，原样输出，尽量先输出该字段
- 保持事件密度合理：赶路日 0 事件，一般日 0-1 事件，关键日 1-3 事件
- 琐事直接描写过掉不给选择；小事给 2-3 个选项快速收束；中事以上展开
- 称呼必须合乎汉末礼制：县令称"明府"，郡守称"使君"，同级称"足下"，长辈称"丈人"
- NPC 基于自身价值观和掌握的信息做出反应，不为剧情方便而扭曲
- 当众与私下反应不同：当众失面子伤害×3

**输出格式：**
严格按照 JSON 格式返回，包含 narration、scene_participants、choices、以及可选的 _system 字段。
不要 JSON 之外的文字（含 ``` 围栏）。
