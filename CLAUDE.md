## Build, run, test

There are no command-line build, lint, or test entrypoints in this repo. Unity projects of this shape are driven through the Editor:

- **Run / Play:** open the project in Unity 2022.3.62f2 and press Play in the Editor.
- **Build:** *File → Build Settings* in the Editor. No build target is currently configured.
- **Tests:** the Unity Test Framework is not installed (no `com.unity.test-framework` in `manifest.json`, no `Tests` assembly definitions). Add the package and create a test asmdef before attempting to run tests.

If a CLI build/test workflow is needed later, it would be invoked via `Unity.exe -batchmode -projectPath ACLS_Unity ...`; do not assume one exists today.
